## 冒泡排序

```c++
void sort(vector<int>& v) {
    int nums = v.size();
    // 无需排序
    if(v.size() <= 1) {
        return;
    }
    // 每一轮，将最大（最小) 的元素移动到最后
    for(int i=0; i<nums; i++) {
        // 两两比较，交换
        for(int j=1; j<nums; j++) {
            if(v[j] < v[j-1]) {
                int t = v[j];
                v[j] = v[j-1];
                v[j-1] = t;
            }
        }
    }
}
// std::vector<int> v = {1, 3, 4, 6, 2, 10, 1};
// 0:  1 3 4 6 2 10 1
// 1:  1 3 4 2 6 1 10 
// 2:  1 3 2 4 1 6 10
// 3:  1 2 3 1 4 6 10
// 4:  1 2 1 3 4 6 10
// 5:  1 1 2 3 4 6 10
```

## 归并排序

### 自顶向下

1. 将一个列表分割为等长2部分，对两个子列表递归进行归并排序，将两个排序后的有序列表合并为一个有序序列
2. 当列表仅剩一个元素时，即为有序列表，直接返回

对一个列表进行排序， 使用归并排序
```c++
struct ListNode {
    int val;
    ListNode *next;
    ListNode() : val(0), next(nullptr) {}
    ListNode(int x) : val(x), next(nullptr) {}
    ListNode(int x, ListNode *next) : val(x), next(next) {}
};

class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if(head == nullptr) {
            return head;
        }
        return sort_list(head, nullptr);
    }

    ListNode* sort_list(ListNode* start, ListNode* end) {
        // 两个元素，分割为1
        if(start->next == end) {
            start->next = nullptr;
            return start;
        }
        // 找中间, 快慢指针
        ListNode* fast = start, *slow = start;
        while(fast != end) {
            fast = fast->next;
            slow = slow->next;
            if(fast != end) {
                fast = fast->next;
            }
        }
        ListNode* middle = slow;
        return merge(sort_list(start, middle), sort_list(middle, end));
    }

    ListNode* merge(ListNode* l1, ListNode* l2) {
        ListNode* n = new ListNode(0);
        ListNode* tmp = n;
        while(l1 != nullptr && l2 != nullptr) {
            if(l1->val < l2->val) {
                tmp->next = l1;
                l1 = l1->next;
            } else {
                tmp->next = l2;
                l2 = l2->next;
            }
            tmp = tmp->next;
        }
        if(l1 != nullptr) {
            tmp->next = l1;
        } else if(l2 != nullptr) {
            tmp->next = l2;
        }
        return n->next;
    }
};
```

## 计数排序

计数排序为桶排序的一种特殊形态， 若待排序的列表为正整数数字，且最大值在一定范围（最大值过大会导致空间复杂度过高），可使用计数排序

- 时间复杂度：`O(数组长度+数组最大值)`
- 空间复杂度：`O(数组最大值)`

计数排序由2步完成：
1. 排序：创建一个最大值长度的缓存数组，其下标即代表0-最大值，对应值代表其出现的次数；遍历原数组，统计每一个值出现的次数
2. 输出：遍历缓存数组，若为0，即代表当前下标对应的数字未出现，若大于0，输出该次数的下标数字

```c++
vector<int> sort(vector<int>& v) {
    if(v.size() <= 1) {
        // 无需排序
        return v;
    }
    int max_value = *max_element(v.begin(), v.end());
    // 创建最大值长度的数组
    vector<int> cache(max_value+1);
    for(auto& i : cache) {
        i = 0;
    }
    // 统计每个元素出现的次数
    for(auto& i : v) {
        cache[i]++;
    }
    // 初始化结果数组
    vector<int> res(v.size())
    // 输出结果
    for(int i=0; i<max_value+1; i++) {
        int t = cache[i];
        while(t > 0) {
            res.push_back(i);
            t--;
        }
    }
    return res;
}
```
