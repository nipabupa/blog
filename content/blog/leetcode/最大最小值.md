## 前缀和

一个数组前N个值的和的数组

```cpp
// 原数组
int nums[100];

// 前缀和数组
int cache[100];
int count = 0
for(int i=0; i<100; i++) {
    count = count + nums[i];
    cache[i] = count;
}

```

例: 找出一个整数数组具有最大和的子数组

解: 遍历计算前缀和, 同时记住最大前缀和与最小前缀和，两个相减(有负数)或最大前缀和即为最大和子数组
