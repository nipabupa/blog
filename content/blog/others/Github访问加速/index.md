## 网易UU加速器

1. [官网下载](https://uu.163.com/)，自行安装
2. 打开软件，注册并登录账号（不需要开会员）
3. 右上角搜索框搜索`学术资源`，点击选择并开启加速
![学术资源](search.png)
4. 浏览器弹出一个界面，忽略
5. 加速器显示加速成功，即可快速访问Github
![加速成功](加速成功.png)
